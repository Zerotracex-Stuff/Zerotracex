Infinix GT10 Pro - Vendor Boot Flashing Guide

By Zerotracex April 01, 2025

Prerequisites

  • Windows PC with USB ports
  • USB cable (preferably the original one)
  • Unlocked bootloader
  • Custom or stock vendor_boot.img file
  • Device must be on XOS 14

Step 1: Prepare Your Device

  1. Go to Settings > About Phone
  2. Tap Build Number 7 times to enable Developer Options
  3. Go to Settings > System > Developer Options
  4. Enable USB Debugging

Step 2: Set Up the Flashing Tool

  1. Download Flashing tool on your PC
  2. Open the tool

Step 3: Connect Your Device and Enter Fastboot Mode

  1. Connect your Infinix GT10 Pro to the PC using a USB cable
  2. On your phone, tap Allow when prompted for USB debugging permission
  3. Press on Check Device
  4. In the tool, check the connection status - it should show Connected via ADB
  5. Click Reboot to Fastboot to switch to fastboot mode
  6. Your device will restart into a special screen with text
  7. Wait for the tool to show "Connected via Fastboot"

Step 4: Select Vendor Boot Image

  1. Click the Browse button in the tool
  2. Navigate to and select your vendor_boot.img file
  3. The tool will verify the file is present
  4. If the file is missing, you'll see an error message

Step 5: Flash Vendor Boot Image

  1. Click the Flash Vendor Boot button
  2. Read the warning message carefully
  3. Click Confirm to proceed
  4. Monitor the flashing process:
    • Progress bar shows overall completion percentage
    • Log window displays detailed operation steps
    • Status label shows current operation
  5. Do not disconnect your device or interrupt the process

Step 6: Complete the Process

  1. Wait for "Process completed successfully" message
  2. Click Reboot Device to restart your phone
  3. Your device should now boot with the new vendor boot image

Troubleshooting

  • If the process fails, try repeating the steps
  • Ensure you're using the correct vendor_boot.img for your device
  • Check the log window for specific error messages
  • If your device fails to boot, you may need to reflash the original vendor_boot.img

WARNING

  • Flashing an incorrect vendor_boot.img may cause boot issues
  • Interrupting the flashing process may brick your device
  • Always keep a backup of your original vendor_boot.img
  • Proceed at your own risk
